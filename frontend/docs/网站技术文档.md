## 登录页面功能

### 文件
- `login.html`（单文件，内含HTML/CSS/JS）

### 功能列表

| 功能 | 说明 |
|------|------|
| 双重登录 | 账号密码 + 手机验证码 |
| Tab 切换 | 两种登录方式切换 |
| 表单验证 | 实时验证 + 提交验证 |
| 图形验证码 | Canvas生成4位验证码，点击刷新 |
| 密码显示/隐藏 | 点击眼睛图标切换 |
| 短信验证码 | 60秒倒计时 |
| 记住我 | 保存用户名到localStorage |
| 登录状态 | 保存token到localStorage |

### 测试账号

| 类型 | 账号 | 密码 |
|------|------|------|
| 账号密码 | admin | 123456 |
| 手机验证码 | 任意有效手机号 | 123456 |

### 登录后行为
1. 保存 `auth_token` 到 localStorage
2. 保存 `user_info` 到 localStorage
3. 跳转到 index.html
4. 首页显示用户名，点击可退出
